STEP 3: MANUAL SETUP (Agar automatic kaam na kare)
================================================================================

TERMINAL 1 - Backend Setup:
---------------------------
1. Terminal/PowerShell kholo
2. Type karo:
   cd backend
   npm install
   
3. .env file banao:
   - backend folder mein "env.example.txt" file kholo
   - Copy karo sab content
   - Nayi file banao ".env" naam se (dot se start)
   - Content paste karo

4. Backend start karo:
   npm start
   
   Success message dikhna chahiye:
   "üöÄ Server running on port 5000"

================================================================================
COMMON ERRORS & SOLUTIONS
================================================================================

Error: "node command not found"
  ‚Üí Node.js install nahi hai
  ‚Üí https://nodejs.org/ se install karo

Error: "Port already in use"
  ‚Üí Port 5000 ya 3000 already use ho rahi hai
  ‚Üí .env file mein PORT=5001 karo (backend ke liye)
  ‚Üí Frontend: Terminal mein set PORT=3001 karo

Error: "npm install failed"
  ‚Üí Internet connection check karo
  ‚Üí npm cache clean --force
  ‚Üí Phir se npm install

Error: "Module not found"
  ‚Üí Correct folder mein ho (backend ya frontend)
  ‚Üí node_modules delete karo
  ‚Üí npm install phir se
